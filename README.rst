About simpletpb
----------------
python + webpy搭建的爬虫，用于镜像海盗湾。 目前运行在sae上。 http://www.simpletpb.com

这是初学Python时的习作。采用web.py，定期爬取海盗湾的资源，并从豆瓣上抓取相应的资源信息匹配。可运行于sae及dotcloud平台上。
